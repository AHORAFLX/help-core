# Instalador de Flexygo Core

El instalador de Flexygo Core te guía para desplegar tu aplicación de la forma más adecuada a tus necesidades.  
Puedes elegir entre tres métodos principales:

- **Instalación Básica:**  
  Todo se instala automáticamente en un único sitio IIS. Ideal para pruebas, demos o entornos pequeños.

- **Instalación Avanzada:**  
  Permite configurar varios sitios IIS, rutas personalizadas, múltiples componentes, etc. Para instalaciones personalizadas en producción.

- **Instalación Docker:**  
  Genera los archivos y configuraciones necesarios para desplegar la aplicación usando contenedores Docker.

---

## ¿Qué opción elegir?

!!! tip "¿No sabes qué método te conviene?"
    - **¿Buscas simplicidad y tienes un único servidor?**  
      Usa la [Instalación Básica](iis.md).
    - **¿Necesitas personalización o balanceo de carga?**  
      Consulta la [Instalación Avanzada](iis.md).
    - **¿Trabajas con contenedores o cloud?**  
      Ve directamente a [Instalación Docker](docker.md).

Cada opción tiene una guía paso a paso específica:

- [Guía detallada para IIS](iis.md)
- [Guía para Kestrel](kestrel.md)
- [Guía de despliegue Docker](docker.md)

---

> Una vez decidas la vía que más te conviene, sigue la guía correspondiente para completar la instalación.
