# Configuración del servidor MCP

El proyecto generado desde la plantilla de Flexygo Core ya incluye el archivo `.vscode/mcp.json` con la configuración necesaria para arrancar el servidor directamente desde VS Code.

```json
{
  "servers": {
    "flexygo": {
      "command": "flexygo-mcp"
    }
  }
}
```

Consulta [Uso básico](uso.md) para ver cómo arrancar el servidor y conectarlo al agente.
