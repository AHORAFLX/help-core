*[IIS]: Internet Information Services — servidor web de Windows
*[DACPAC]: Data-tier Application Package — archivo de despliegue de esquema SQL Server
*[NuGet]: Gestor de paquetes para .NET
*[SDK]: Software Development Kit
*[Application Pool]: Grupo de aplicaciones de IIS que aísla procesos web
*[SMTP]: Simple Mail Transfer Protocol — protocolo de envío de correo
*[CORS]: Cross-Origin Resource Sharing — política de seguridad del navegador para peticiones entre dominios
*[JWT]: JSON Web Token — estándar para tokens de autenticación
*[MCP]: Model Context Protocol — protocolo para conectar herramientas IA con contexto externo
*[CI/CD]: Integración Continua / Despliegue Continuo
*[STARTTLS]: Extensión del protocolo SMTP que inicia cifrado TLS en conexiones en texto plano
